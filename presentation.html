Feedback Collector

Simple customer feedback system for small businesses

Mark Mugerman

m.mugerman@innopolis.university

Group: CSE-01

Context

👤 End User

Small business owners — café operators, bakery managers, repair shop owners — who need to track customer reviews without complex tools.

⚠️ Problem

Customer feedback is scattered across verbal comments, social media DMs, and paper notes — making it hard to spot issues or praise systematically.

💡 Solution

A simple feedback collector: customers submit comments via a web form with star rating, and the business owner sees all feedback on a clean dashboard with statistics, bar charts, and filtering.

Implementation

Version 1 — Core Feature

  • Feedback form with 1–5 star rating
  • REST API (Node.js + Express)
  • JSON file-based database
  • Owner dashboard (newest first)
  • Responsive Bootstrap UI

Version 2 — Additional Features

  • 📊 Bar chart for rating distribution
  • Filter by rating (keeps newest first)
  • Automatic email notifications
  • Statistics cards (total, average, 5★)
  • Docker + docker-compose deployment
  • 26/26 API integration tests passed
